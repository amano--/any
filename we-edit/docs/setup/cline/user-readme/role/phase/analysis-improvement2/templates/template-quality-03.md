## 2. チェックリストテンプレート

## チェック項目

□ データの網羅性確認
□ データの正確性確認
□ データの一貫性確認
□ 欠損値の処理確認

### 手法チェック
□ 分析手法の適切性
□ パラメータの妥当性
□ 前提条件の確認
□ 制約条件の確認

### 結果チェック
□ 結果の妥当性
□ 解釈の論理性
□ 限界の認識
□ 提言の実現性
