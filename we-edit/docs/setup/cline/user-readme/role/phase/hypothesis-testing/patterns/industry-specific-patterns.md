# 業界別仮説検証パターン集

## 1. EC/小売業界

### 1.1 主要な仮説パターン
```markdown
### 顧客行動仮説
1. 購買意思決定
   - 商品検索プロセス
   - 比較検討行動
   - カート追加判断
   - 購入完了要因

2. 離脱要因
   - 検索離脱
   - カート離脱
   - 決済離脱
   - リピート阻害

### 価格設定仮説
1. 価格感度
   - 価格帯別反応
   - セール効果
   - 送料影響
   - バンドル効果

2. 競争環境
   - 競合比較
   - 差別化要因
   - 価格優位性
   - ブランド影響
```

### 1.2 検証アプローチ
```markdown
### 定量的検証
1. A/Bテスト設計
   - 価格変動テスト
   - UI/UXテスト
   - 商品表示テスト
   - プロモーション効果

2. コホート分析
   - 顧客セグメント別
   - 購入頻度別
   - 平均購入単価別
   - 定着率分析

### 定性的検証
1. ユーザーインタビュー
   - 購入プロセス確認
   - 意思決定要因
   - 不満点ヒアリング
   - 改善要望収集

2. 行動観察
   - ナビゲーション行動
   - 検索パターン
   - 比較検討プロセス
   - 離脱ポイント
```
