# 課題検証テンプレート

## 1. 課題仮説フレームワーク

### 1.1 基本情報
```markdown
仮説ID: [ID番号]
作成日: [YYYY-MM-DD]
作成者: [名前]
対象ユーザー: [具体的なペルソナ]
```

### 1.2 課題仮説の定義
```markdown
### 観察された現象
[具体的な観察事実や指標]

### 推測される課題
[ユーザーが直面していると考えられる問題]

### 仮説の根拠
- [データポイント1]
- [データポイント2]
- [定性的な観察]

### 検証が必要な不確実性
1. [不確実性1]
2. [不確実性2]
3. [不確実性3]
```
