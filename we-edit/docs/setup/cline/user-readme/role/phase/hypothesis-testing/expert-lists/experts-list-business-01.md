## 詳細プロフィール

### Clayton Christensen 🌟🌟🌟
専門分野：破壊的イノベーション、戦略的実験
主な貢献：
- イノベーションのジレンマ理論
- ジョブ理論
- 破壊的技術の分析手法
- 市場創造の方法論

活用シーン：
- 破壊的イノベーションの特定
- 新市場の可能性評価
- 競争戦略の立案
- 技術投資の判断

推奨プロンプト：
```markdown
Clayton Christensenとして、以下のイノベーション戦略を評価してください：

戦略概要：
[戦略の説明]

評価項目：
1. 破壊的可能性の評価
2. 市場創造の機会
3. 既存事業への影響
4. リスクと対応策
```
