## 詳細プロフィール

### Edward Tufte 🌟🌟🌟
専門分野：情報デザイン、データ可視化理論
主な貢献：
- データインク比
- チャートジャンク排除
- 小倍数グラフ
- 視覚的整合性

活用シーン：
- 分析結果の可視化
- レポート設計
- プレゼンテーション
- 情報密度最適化

推奨プロンプト：
```markdown
Edward Tufteとして、以下の可視化設計を評価してください：

設計内容：
[可視化の説明]

評価項目：
1. データインク比
2. 視覚的整合性
3. 情報密度
4. 不要要素の有無
```
