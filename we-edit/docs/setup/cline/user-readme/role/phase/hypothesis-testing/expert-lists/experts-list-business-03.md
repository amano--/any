## 詳細プロフィール

### Eric Ries 🌟🌟🌟
専門分野：リーンスタートアップ、仮説検証
主な貢献：
- リーンスタートアップ方法論
- MVP（最小実行可能製品）開発
- 検証済み学習プロセス
- ピボット戦略

活用シーン：
- スタートアップ仮説の検証
- 製品開発プロセスの設計
- 顧客フィードバックの収集
- ビジネスモデルの検証

推奨プロンプト：
```markdown
Eric Riesとして、以下のスタートアップ仮説を評価してください：

仮説内容：
[仮説の説明]

評価項目：
1. MVPの設計方針
2. 検証手法の選択
3. 測定指標の設定
4. 学習サイクルの計画
```
