# フェーズ別資料作成ガイド - 概要

## 1. 目的と範囲

本ガイドは、各開発フェーズにおける資料作成のための包括的なガイドラインを提供します。

### 1.1 主な目的
- フェーズ別の標準的な資料構成の提示
- 効果的な資料作成プロセスの確立
- 品質と一貫性の確保
- 再利用可能な知見の蓄積

### 1.2 対象範囲
- ディレクトリ構造とファイル命名
- 専門家リストの作成
- テンプレートの設計
- 活用ガイドの作成
- 事例集の整備
- メンテナンス手順
- トラブルシューティング

## 2. ガイドの構成

### 2.1 ドキュメント構成

## 3. 使用方法

### 3.1 初回利用時
1. 00-overview.mdで全体像を把握
2. 01-structure.mdでディレクトリ構造を確認
3. 必要な資料に応じて各ガイドを参照

### 3.2 資料作成時
1. 対象フェーズの特徴を確認
2. 必要な専門家リストを作成
3. テンプレートを準備
4. 活用ガイドを整備

### 3.3 メンテナンス時
1. 定期レビューの実施
2. フィードバックの収集
3. 改善の実施
4. 文書の更新
