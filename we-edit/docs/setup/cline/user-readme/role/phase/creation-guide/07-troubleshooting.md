# エキスパートロールのトラブルシューティング

## 1. 一般的な問題と解決策

### 1.1 ロール設定の問題
```markdown
1. 専門性が不明確
   □ 症状
      - 出力が一般的すぎる
      - 専門知識が反映されない
      - 特徴的な視点が欠ける
      - アドバイスが表面的

   □ 解決策
      - 専門分野の具体的な指定
      - 実績の明確な言及
      - 特徴的手法の強調
      - 具体的な評価基準の設定

2. 組み合わせの不調和
   □ 症状
      - 矛盾する助言
      - 重複する視点
      - 連携不足
      - 統合の困難さ

   □ 解決策
      - 役割分担の明確化
      - 優先順位の設定
      - 連携方法の具体化
      - 統合プロセスの確立
```

### 1.2 出力品質の問題
```markdown
1. 品質の不均一
   □ 症状
      - レベルのばらつき
      - 一貫性の欠如
      - 深さの不足
      - 具体性の不足

   □ 解決策
      - 品質基準の明確化
      - チェックリストの活用
      - レビュープロセスの強化
      - フィードバックの反映

2. コンテキスト理解不足
   □ 症状
      - 不適切な提案
      - 状況にそぐわない助言
      - 制約考慮の不足
      - 現実性の欠如

   □ 解決策
      - 背景情報の充実
      - 制約条件の明確化
      - 目的の具体化
      - 期待値の調整
```
