# 役割別ガイドライン

## 1. 文書作成者向け

### 1.1 初級者向け
```markdown
# 初級者ガイド

## 基本的な作業
1. テンプレートの使用
   - 適切なテンプレートの選択
   - 必須項目の入力
   - 基本的な形式の維持
   - チェックリストの活用

2. 品質確保の基本
   - スペルチェック
   - リンク確認
   - フォーマット確認
   - 基本的なレビュー依頼

## 推奨プラクティス
1. 段階的なアプローチ
   - テンプレート選択から開始
   - 基本情報の入力
   - 内容の段階的な追加
   - レビュー依頼と修正

2. スキル向上のステップ
   - 基本ルールの習得
   - テンプレートの理解
   - ツールの基本操作
   - フィードバックの活用
```

### 1.2 中級者向け

```markdown
# 中級者ガイド

   - ショートカットの活用

   - ベストプラクティスの適用

2. 品質の向上
   - チェックリストの拡充
   - レビュー方法の改善
   - フィードバックの活用

```
